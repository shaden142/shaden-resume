Course Project — CCCY431 Machine Learning

ML-Based Phishing Website Detection

Trained and compared five classifiers (Logistic Regression, Decision Tree, Random Forest, SVM, KNN) on a 651K-URL dataset, engineering 22 features from domain, structural, lexical, and security-indicator patterns.

Random Forest came out on top, and I built a real-time detection interface with Streamlit so a URL can be checked instantly.

Enterprise Network Security: Red & Blue Team Simulation

Built a full enterprise Active Directory environment, then ran an APT-style Red Team attack chain — LLMNR poisoning, Kerberoasting, BloodHound privilege mapping, up to a Golden Ticket domain compromise.
